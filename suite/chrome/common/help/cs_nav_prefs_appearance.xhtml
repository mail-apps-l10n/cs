Vzhled

Tato sekce popisuje nastavení předvoleb kategorie Vzhled. Dialogové okno pro nastavení předvoleb získáte tak, že:

  1. Otevřete nabídku &brandShortName; Úpravy a klepněte na položku Předvolby.
  2. Klepněte na kategorii Vzhled.

Kategorie Vzhled vám umožňuje nastavit některé aspekty chování při startu balíku &brandShortName;:

  • Po spuštění se otevře: Zde můžete zvolit komponenty, které chcete, aby se otevíraly při startu balíku &brandShortName;
  • Zobrazit nástrojovou lištu jako:
    • Obrázky a text: Navigační lišta bude obsahovat jak tlačítka s obrázky, tak jejich popisy.
    • Pouze obrázky: Navigační lišta bude obsahovat pouze obrázky.
    • Pouze text: Navigační lišta bude obsahovat pouze text.
  • Zobrazovat plovoucí tipy: Zaškrtněte, pokud chcete povolit zobrazování tooltips při ponechání kurzoru myši nad uživatelským prvkem rozhraní s definovanou místní nápovědou.
  • Jazyk uživatelského rozhranní: This setting allows you to change the language used in the user interface of &brandShortName;. Additional languages can be installed from the &brandShortName; home page. Poznámka: You must restart &brandShortName; for a new language setting to take effect.

Obsah

Tato sekce popisuje nastavení předvoleb kategorie Obsah. Dialogové okno pro nastavení předvoleb získáte tak, že:

  1. Otevřete nabídku &brandShortName; Úpravy a klepněte na položku Předvolby.
  2. V kategorii Vzhled klepněte na podkategorii Obsah. (Pokud nevidíte žádnou podkategorii, dvojklepnutím na Vzhled zobrazíte jejich seznam.)

The Content preferences panel allows you to change settings that influence how website and message content appears in &brandShortName;.

  • Ikonky webového serveru:
    • Zobrazovat ikony webového serveru: Select this if you want see site-specific icons, if available, in place of the bookmark icon . Website icons are shown to the left of the Location Bar and Browser tabs.
    • Automaticky hledat ikonu webového serveru, pokud není ve stránce uvedena: If the page itself does not define a website icon, turning on this setting makes &brandShortName; look for a "favicon" on the server and use that instead.
    • Zobrazovat ikony webového serveru u záložek v nabídce a na liště:
      • Nikdy u záložek nezobrazovat ikony webového serveru: Select this if you only want to see the default icons but not the website's own icon in the bookmarks menu or the personal toolbar.
      • Pouze pokud byla stránka nedávno navštívena: Select this to show the website's own icon only if the website has been recently loaded and the icon is currently in the browser's cache.
      • Vždy u záložek nahrávat ikony webového serveru: Select this to always load website icons to be displayed in the bookmarks menu or personal toolbar, even if it's not in the cache.
  • Použít jemné posunování: Zvolte tuto volbu pro povolení jemného posunování. Stiskem klávesy Page Down budete postupně přesunuty na další stránku místo okamžitého skoku.
  • Povolit automatickou změnu velikosti obrázků: Zvolte tuto volbu, pokud chcete, aby balík &brandShortName; automaticky zmenšoval velikosti obrázků na velikost okna tehdy, pokud je tento obrázek větší. Klepnutím na takto zmenšený obrázek jej zvětšíte do plné velikosti.
  • Měnit velikost pouze textu místo celé stránky: Select this if you want &brandShortName; to only resize text of websites when using the "Zoom" function. If this is not selected, the whole page, including images, will be zoomed.

Poznámka: See the separate preferences panels for colors, fonts and languages to further customize content appearance and the Privacy & Security section for privacy-related settings that also might influence how content appears to you.

Písma

Tato sekce popisuje nastavení předvoleb v podkategorii Písma. Dialogové okno pro nastavení předvoleb získáte tak, že:

  1. Otevřete nabídku &brandShortName; Úpravy a klepněte na položku Předvolby.
  2. V kategorii Vzhled klepněte na podkategorii Písma. (Pokud nevidíte žádnou podkategorii, dvojklepnutím na Vzhled zobrazíte jejich seznam.)

Podkategorie Písma vám umožňuje nastavit různé druhy a velikosti písma pro zobrazení webových stránek.

Poznámka: Výběr fontů některých písmen může být zablokován z důvodů neexistence fontu pro dané písmo a znakové sady.

  • Znaková sada: Zvolte znakovou sadu, která odpovídá vašemu umístění (pro Českou republiku Středoevropské jazyky). Pro skupinu jazyků, která není v seznamu, zvolte Definovaná uživatelem.
    • Proporcionální: Zvolte, zda chcete používat patkové či bezpatkové písmo. Můžete také použít různou velikost písma pro proporcionální text. Znaky proporcionálního textu mají proměnlivou šířku.
    • Patkové (Serif): Zvolte název fontu pro patkové písmo.
    • Bezpatkové (Sans-serif): Zvolte název fontu pro bezpatkové písmo.
    • Kurzíva (Cursive): Zvolte název fontu pro kurzívu.
    • Ozdobné (Fantasy): Zvolte název fontu pro ozdobné písmo.
    • Neproporcionální (Monospace): Zvolte název s velikostí fontu pro neproporcionální písmo. Znaky neproporcionálního písma mají stejnou šířku, zabírají tedy stejné místo.
    • Minimální velikost fontu: Pokud chcete, aby prohlížeč používal písma od určité velikosti, vyberte hodnotu z rozbalovacího seznamu.
  • Povolit stránkám použít jiné písmo: Zvolením zaškrtávacího políčka povolíte prohlížeči používat velikost a druh písma určené autorem webové stránky.

Barvy

Tato sekce popisuje nastavení předvoleb v podkategorii Barvy. Dialogové okno pro nastavení předvoleb získáte tak, že:

  1. Otevřete nabídku &brandShortName; Úpravy a klepněte na položku Předvolby.
  2. V kategorii Vzhled klepněte na podkategorii Barvy. (Pokud nevidíte žádnou podkategorii, dvojklepnutím na Vzhled zobrazíte jejich seznam.)

Podkategorie Barvy vám umožňuje nastavit výchozí barvy pro text a pozadí webových stránek:

  • Text a pozadí: Klepnutím na čtvereček s aktuální barvou textu a pozadí vyberete nové barvy zobrazované na webových stránkách.
    • Použít systémové barvy: Zaškrtněte, pokud chcete použít výchozí nastavení barev vašeho operačního systému.
  • Barvy odkazů: Klepněte na čtvereček s aktuální barvou odkazů pro vybrání barvy, jenž se bude zobrazovat u navštívených, active, a nenavštívených odkazů webových stránek.
    • Podtrhávat odkazy: Zvolte, pokud chcete text odkazu podtrhávat.
  • Pokud stránka nabízí vlastní barvy a pozadí:
    • Vždy použít barvy a pozadí specifikované na stránce: Umožňuje zobrazit barvy a pozadí specifikované autorem webové stránky.
    • Vždy použít mnou vybrané barvy, potlačit barvy nastavené na stránce: Umožňuje ignorovat barvy a pozadí specifikované autorem webové stránky.
